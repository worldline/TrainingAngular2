Ecosystem About Typescript - Is it required?

  • Short answer: No
  • Typescript is not specific to Angular
  • the typescript code is just transpiled in javascript (from ES3 to ES6)
  • So you can use javascript to develop you Angular2 applications as well
  • But: typescript is used to develop the Angular2 framework
  • It has even been developed in conjunction with Angular2
  • It is a typed language
  • Which is close to recent JavaScript syntax
  • Which allows OOP
  • Which allows to easily integrate recents best practices such as js modules loading
  • And is interesting as soon as a project is big enough
