Ecosystem Typescript - Why to use it

Typescript is an open source Microsoft project. It allows to add typings constraints upon the javascript syntax. Again, typescript is not mandatory, you can write in pure ES6 js as well if you want. But here are a few reasons we recommend it:

  • For big projects, typed langages will help as well as for developer tools or for making complex Conception Oriented Object models
  • More specifically, for Angular2 apps, it will be particularly interesting for DI mechanism which will be naturally based on types to automatically inject the needed data at some point...
  • Angular2 itself is based on typescript so you will have access to all types definitions
  • Learning typescript is not much more than learning ES6
