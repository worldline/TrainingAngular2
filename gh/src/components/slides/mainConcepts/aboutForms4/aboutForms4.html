Main Concepts Forms - FormControl and FormGroup

  • Based on the previous sample, we can
    • Assign a control to an input
    • Then play with CSS: for example use the ng-invalid class to set red borders on this element
    • Check the validity (or some other properties) of a control in order to dynamise the UI
  • It's also possible to attach your control(s) to a parent by using the FormGroup object
  • This is interesting to get properties for the whole form: for example the group is valid if each of its controls are valid...
  • The FormBuilder API help us creating a group and its control

Below, let's disable the submit button if the form is not valid

import {Component} from '@angular/core'; import {BrowserModule} from '@angular/platform-browser'; import {FormsModule, ReactiveFormsModule} from '@angular/forms'; import {MyFormCmp} from 'app/my-form.component'; @Component({ selector: 'app', template: '<my-form-cmp></my-form-cmp>' }) export class App {} export const ModuleDefs= { bootstrap: [App], imports: [BrowserModule, FormsModule, ReactiveFormsModule], declarations: [App, MyFormCmp] } import {Component} from '@angular/core'; import {FormBuilder, FormGroup, FormControl, Validators} from '@angular/forms'; enum TodoCategory{PRIVATE, PERSO}; class TodoModel{ constructor(public category: TodoCategory, public details: string){} } @Component({ selector:'my-form-cmp', templateUrl:'app/my-form.html' }) export class MyFormCmp{ categories: string[]= Object.keys(TodoCategory).filter(v => isNaN(parseInt(v, 10))); model: TodoModel= new TodoModel(); formGroup: FormGroup; constructor(fb: FormBuilder){ this.formGroup = fb.group({ 'detailsControl': new FormControl('', Validators.required) }); } isSubmit: false; onSubmit(){ this.isSubmit= true; } } <h1>Add an element to your todo list</h1> <form (submit)="onSubmit()"> <p> <label for="category"> Category </label> <select name="category" [(ngModel)]="model.category" id="category" required> <option *ngFor="let category of categories" [value]="category">{{category}}</option> </select> </p> <p> <label for="todoDetails">Content</label> <input name="details" [(ngModel)]="model.details" [formControl]="formGroup.controls.detailsControl" #currentElt > <span *ngIf="!formGroup.controls.detailsControl.valid">Miss data here</span> </p> <button *ngIf="formGroup.valid" type="submit">Submit</button> </form> <p> {{currentElt.classList}} </p> <p *ngIf="isSubmit"> Form submited </p> <h2>Result</h2> <p *ngIf="!isSubmit"> {{model.category}} : {{model.details}} </p>
