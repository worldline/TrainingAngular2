Main Concepts About Templates - Event Binding

Listening events is a common need in javascript

  • With ng2, this is a native behavior, and you don't need any new devlopment to manage new events
  • Just use (eventToManage)= "expression"
  • Don't like the parenthesis? Then use the on-eventname = "expression" syntax instead
import {Component} from '@angular/core'; import {BrowserModule} from '@angular/platform-browser'; @Component({ selector:'app', templateUrl:'app/app.html' }) class AppComponent{ logEvent(msg, event){ console.log(msg); console.log(event); } } export const ModuleDefs= { imports: [BrowserModule], declarations: [AppComponent], bootstrap:[AppComponent] } <h1>Checking event handling</h1> <p (click)="logEvent('click', $event)" > Click Me And check the logs </p>
