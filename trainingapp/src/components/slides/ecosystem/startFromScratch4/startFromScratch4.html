Ecosystem Start From Scratch - polyfills and libs

Polyfills are scripts which improve browsers abilities. For example, if you decide to use Babel to transpile your ES6 at runtime, you're using a polyfill.

  • zone.js is a library which allows to listen events and is used by the Angular2 detection change mechanism
  • reflect is used in to add metadata into ES5, which is needed because Angular2 uses annotations to add metadata
  • typescript allows you to choose the syntax you want for modules (AMD, CommonJS and some others). Once generated, your browser will need a polyfill to understand this syntax. Using SystemJs is a common way to do this

So to import needed libraries/polyfills, a minimal index html page imports could look like this:

<!-- polyfills and libs --> <script src="https://unpkg.com/zone.js@0.6.12"></script> <script src="https://unpkg.com/reflect-metadata@0.1.3"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/systemjs/0.19.27/system.js"></script>
