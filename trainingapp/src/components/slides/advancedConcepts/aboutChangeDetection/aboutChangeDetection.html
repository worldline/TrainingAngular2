Advanced Concepts Change Detection - Intro

  • Change Detection is an important Angular2 feature
  • And strategic also
  • Because it can be costly, as it was with Angular1 and its digest cycle
  • The idea is to let the framework automatically update the view when something changes in the model
  • As an example, you want to display the user name in your view.
    • You use the $http service to get the name of the user based on its login
    • On the $http answer, you update the field name with the received value
    • Name which is used in your view through the expansion {{name}}
    • And this works! But how Angular2 is aware of the name field update?
