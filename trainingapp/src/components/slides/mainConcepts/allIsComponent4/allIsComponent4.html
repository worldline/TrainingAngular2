Main Concepts Classes Are Everywhere - Pipes

Pipes are the equivalent to filters in Angular1, they allow to apply a treatment to an entry value, eventually with parameters, and return a value.

/** * Example: * {{ 'a string' | truncate:3 }} */ @Pipe({ name: 'truncate' }) class TruncatePipe implements PipeTransform{ private defaultIndex = 4; transform(val: string, index?: number) { var idxToUse = (index === undefined ? this.defaultIndex : index); return val.substr(0, idxToUse); } }
