Main Concepts PW - Forms and validators

Let's create our custom validator

  • Browse to the Contact component
  • Use the appropriate directive to get 2 way data binding between your textarea content and a field in your Contact class
  • Add a form element arround your textarea and send button
  • Use the FormBuilder api to create a FormGroup containing a custom control for the message. The control has to contain 2 validators:
    • required
    • a new custom validator which disables the submit button while the message contains any of these characters: \ < > & #
